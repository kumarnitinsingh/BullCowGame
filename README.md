# BullCowGame
This is bullcow game made with unreal engine.

Steps to see this output of project;
1:  clone this repo;
2:  download UnrealEngine on your PC;
3:  click on BullCowGame.uproject;
4:   a window will pop up to rebuild. just click ok or build it will create new binaries;
5:   Congratulations you opened this bullcowgame over unrealEngine editor;just click on play;
